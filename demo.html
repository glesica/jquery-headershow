jQuery-headershow Demo Page

This page demonstrates a simple use of jQuery-headershow.

This is a demo of the jQuery-headershow plugin. It is a simple plugin for creating a nice little slideshow.

It uses elements from the page to create the slideshow, so you don't have to create special data.

Since each slide is taken right from the page, you can use the content twice. Once as a feature in the slideshow, and again within the page itself.

To do this, you just set the hidecontent option to false when creating the slideshow (see page source).

The actual file names of the images do not matter one bit since the content is pulled straight from the DOM.

There are several other options you can set to affect the behavior of the slideshow. You can also add CSS to change the look and feel.
